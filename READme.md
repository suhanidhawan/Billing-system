# 🧾 Billing System (Python Tkinter)

This is a desktop-based Billing System application built using Python and Tkinter. It allows users to generate bills, manage items, and perform basic billing operations through a simple graphical interface.

## 🚀 Features

* Add items with price and quantity
* Automatic bill calculation
* User-friendly GUI
* Clear and generate bills
* Simple and lightweight application

## 🛠️ Tech Stack

* Python
* Tkinter (GUI library)

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/suhanidhawan/Billing-system.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Billing-system
   ```

3. Run the application:

   ```bash
   python main.py
   ```
